# 数据库课设

数据库课程设计项目，包含前端页面、数据库脚本、初始化数据和课程设计相关材料。

## 主要内容

- `index.html` / `styles.css` / `app.js`：前端页面与交互逻辑
- `schema.sql`：数据库表结构
- `seed_100.sql`：示例数据
- `upgrade_db.sql` / `upgrade_college_teacher.sql`：数据库升级脚本
- `server/`：后端或服务端相关代码
- `依赖清单.md`：项目依赖说明

## 使用建议

先导入 `schema.sql`，再按需要执行种子数据和升级脚本。前端页面可直接打开，也可以配合本地服务端运行。

## 安全说明

仓库已忽略本地环境变量、依赖目录、数据库文件、缓存和打包产物。
