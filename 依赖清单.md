# 项目依赖与引用清单

## 一、运行环境

| 名称 | 版本要求 | 用途 |
|------|---------|------|
| Node.js | ≥ 16.x | 后端运行环境 |
| MySQL | 8.x | 数据库服务 |
| 浏览器 | Chrome / Edge 等现代浏览器 | 前端页面访问 |

## 二、后端依赖（NPM 包）

安装位置：`server/node_modules/`，由 `server/package.json` 管理。

| 包名 | 版本 | 协议 | 用途 |
|------|------|------|------|
| express | ^5.2.1 | MIT | Node.js Web 框架，提供路由和中间件能力 |
| mysql2 | ^3.22.1 | MIT | MySQL 数据库驱动，支持 Promise 和连接池 |
| cors | ^2.8.6 | MIT | 处理跨域请求的中间件 |

## 三、前端依赖（CDN 引用）

| 库名 | CDN 地址 | 版本 | 用途 |
|------|---------|------|------|
| ECharts | `https://cdn.jsdelivr.net/npm/echarts@5/dist/echarts.min.js` | 5.x | 数据可视化图表（饼图、柱状图） |

> 除 ECharts 外，前端无其他第三方依赖，全部使用原生 HTML + CSS + JavaScript 实现。

## 四、数据库对象清单

### 4.1 数据表（4张）

| 表名 | 说明 | 关键约束 |
|------|------|---------|
| `students` | 学生信息表 | 主键 `id`，唯一索引 `uk_student_no(student_no)` |
| `courses` | 课程信息表 | 主键 `id`，唯一索引 `uk_course_code(course_code)` |
| `grades` | 成绩记录表 | 主键 `id`，唯一索引 `uk_student_course(student_id, course_id)`，外键级联 |
| `grade_audit_log` | 成绩变更审计日志 | 主键 `id`，索引 `idx_grade_id`、`idx_operated_at` |

### 4.2 触发器（5个）

| 触发器名 | 时机 | 事件 | 目标表 | 作用 |
|---------|------|------|--------|------|
| `trg_calc_grade_point` | BEFORE | INSERT | grades | 录入成绩时自动计算绩点 |
| `trg_update_grade_point` | BEFORE | UPDATE | grades | 修改成绩时自动重算绩点 |
| `trg_audit_grade_insert` | AFTER | INSERT | grades | 录入成绩后写入审计日志 |
| `trg_audit_grade_update` | AFTER | UPDATE | grades | 修改成绩后写入审计日志 |
| `trg_audit_grade_delete` | AFTER | DELETE | grades | 删除成绩后写入审计日志 |

### 4.3 视图（2个）

| 视图名 | 用途 | 涉及表 |
|--------|------|--------|
| `v_student_gpa_stats` | 统计每位学生的加权GPA、已修课程数、总学分 | students + grades + courses |
| `v_course_grade_overview` | 统计每门课程的平均分、最高分、最低分、及格率 | courses + grades |

### 4.4 存储过程（2个）

| 过程名 | 参数 | 用途 |
|--------|------|------|
| `sp_student_transcript(p_student_id)` | 学生ID | 查询某学生的完整成绩单（含等级判定） |
| `sp_course_analysis(p_course_id)` | 课程ID | 统计某课程的成绩五级分布及及格率 |

### 4.5 索引

| 表名 | 索引名 | 类型 | 字段 |
|------|--------|------|------|
| students | PRIMARY | 主键 | id |
| students | uk_student_no | 唯一 | student_no |
| courses | PRIMARY | 主键 | id |
| courses | uk_course_code | 唯一 | course_code |
| grades | PRIMARY | 主键 | id |
| grades | uk_student_course | 唯一 | student_id, course_id |
| grades | idx_course_id | 普通 | course_id |
| grade_audit_log | PRIMARY | 主键 | id |
| grade_audit_log | idx_grade_id | 普通 | grade_id |
| grade_audit_log | idx_operated_at | 普通 | operated_at |

### 4.6 外键约束

| 约束名 | 子表 | 父表 | 策略 |
|--------|------|------|------|
| `fk_grade_student` | grades.student_id → | students.id | ON DELETE CASCADE, ON UPDATE CASCADE |
| `fk_grade_course` | grades.course_id → | courses.id | ON DELETE CASCADE, ON UPDATE CASCADE |

## 六、绩点计算公式

触发器中使用的绩点算法：

```
如果 score ≥ 60：grade_point = (score - 50) / 10
如果 score < 60：grade_point = 0
```

示例：85分 → 绩点 3.5，72分 → 绩点 2.2，55分 → 绩点 0

GPA 视图中的加权平均绩点公式：

```
加权GPA = Σ(各科绩点 × 各科学分) / Σ(各科学分)
```
